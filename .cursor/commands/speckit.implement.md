# Execute Implementation

This command executes the implementation based on the tasks and specifications.

## Purpose
Actually implement the code, components, and features according to the specification and task list.

## Instructions
When this command is invoked, you should:
1. Review the task list and current task
2. Review the specification and implementation plan
3. Review the project constitution for principles and constraints
4. Implement the code following:
   - Project principles and standards
   - Specification requirements
   - Best practices for the technology stack
   - Accessibility guidelines
   - Performance requirements
5. Create or modify files as needed
6. Ensure code quality, proper error handling, and edge cases
7. Add appropriate comments and documentation
8. Follow the project's coding standards and patterns

## Output
Implement the actual code, components, and features according to the established specifications and plans.
