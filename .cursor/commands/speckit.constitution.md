# Establish Project Principles

This command establishes the foundational principles, constraints, and guidelines for the project.

## Purpose
Define the core principles, design philosophy, technical constraints, and project standards that will guide all development decisions.

## Instructions
When this command is invoked, you should:
1. Analyze the current project structure and context
2. Establish clear project principles covering:
   - Design philosophy and UI/UX guidelines
   - Technical architecture principles
   - Code quality standards
   - Performance requirements
   - Accessibility requirements
   - Security considerations
3. Document these principles in a clear, actionable format
4. Create or update project documentation with these principles

## Output
Create a comprehensive project constitution document that serves as the foundation for all subsequent development work.
